Flexbox

Flexbox is a css framework that allows you to position elements without knowing the exact dimensions of your screen width and height. Unlike absolute positioning, it is a responsive framework that can scale or adjust based on varying screen resolutions. For example, you could:

  • Display 3 buttons in a row horizontally with a 10px gap between each with last button pressing the right margin

  • Display images as thumbnails positioned in a grid pattern
     
     
     
     
     
     
     
     
  • Display a list of links as a column on left side of page
    Link 1Link 2Link 3

Basically you can use flexbox to position elements either vertically or horizontally. This is done by setting up an element that acts as a container. All of the children elements of that container can then be positioned vertically or horizontally with various justification and alignment options. This is very simplistic and limited in usefulness until you understand the following rule:

The container element marked as flex only affects or positions its DIRECT children, not all descendents.

Let's look at the example below. The parent has styling of display: flex;. 

<style>
.parent {
    display: flex;
}
</style>

<div class="parent"> <!-- I am the container-->
    <div>I am affected by flex</div>
    <div> I am affected by flex
        <div>I am not affected</div>
        <div>I am not affected</div>
        <div>I am not affected</div>
    </div>
    <div>I am affected by flex</div>
    <div>I am affected by flex</div>
</div>

Again it seems very simplistic, but the rules of Flexbox build on top of this simplicity to make a powerful layout tool. That being said, Flexbox will solve about ~80%-90% of your problems. For those situations where Flexbox is not an easy solution, Flexbox becomes extremely frustrating.

The general idea of flex is simple, but there are many underlying options available for you. Carefully study and use the following link for reference: https://css-tricks.com/snippets/css/a-guide-to-flexbox

Real world example

So how do you use it? First you need to break your layout into a series of cells that can be organized as columns or as rows. Let's take a look at an example. This is a snippet from the NASA MARS ROVER page. Let's focus on recreating the white section.

Let's mark out the entire section with a div. 


<style>
.parent {
    margin: 50px; /* Adding margin to make it easier to see */
}
</style>
    
<div class="parent">
</div>

Now let's look for any rows or columns that we split the picture into. As you can see, we can cleanly separate the page into two segments stacked on top of each other. Remember, flexbox only has an effect on the children of a tag. That being said, the two CHILDREN tags of the parent tag should be oriented in a column format.

Let's set up our children tags and then orient them in a column.


<style>
.parent {
    margin: 50px; /* Adding margin to make it easier to see */
    display: flex;
    flex-direction: column;
}
</style>
    
<div class="parent">
    <div>Rover Update</div>
    <div></div>
</div>

This is what we have so far.

Rover Update

Let's center the top text. And modify the "title" div to have 30px of top padding. This does 2 things:

  1. Fills out space
  2. Pushes the text down

<style>
.parent {
    display: flex;
    flex-direction: column;
}

.title {
    text-align: center;
    font-size: 1.5em;    
}
</style>
    
<div class="parent">
    <div class="title">Rover Update</div>
    <div></div>
</div>
